# DocuMind v0.1.6 릴리스 리포트

- **날짜**: 2026-03-10
- **버전**: 0.1.5 → 0.1.6
- **빌드**: DocuMind_0.1.6_x64-setup.exe / DocuMind_0.1.6_x64_en-US.msi

---

## 변경 사항

### 1. AI 질문 동적 생성 (개선 — 핵심 변경)

**이전**: 문서 유형별 고정 4~5개 질문 (모든 기획서에 같은 질문, 모든 보고서에 같은 질문)
**이후**: AI가 사용자의 입력 주제를 분석하여 맥락에 맞는 질문을 실시간 생성

예시:
- "신규 프로젝트 기획서" → "어떤 분야의 프로젝트인가요?", "대상 시장은?", "경쟁 제품이 있나요?" 등
- "3월 영업 실적 보고서" → "영업 실적 수치는?", "목표 대비 달성률은?", "주요 거래처 변동 사항은?" 등

| 파일 | 변경 내용 |
|------|-----------|
| `src/services/ai-provider/documentAI.ts` | `getRequirements()` (고정 질문) 제거 → `generateQuestions()` (AI 동적 생성) 신규. `QUESTIONS_BY_TYPE` 하드코딩 데이터 전면 삭제. `generateDocument()`에서도 고정 질문 참조 제거 |
| `src/components/ai/AIAssistant.tsx` | `handleTopicSubmit()`을 async로 변경. AI 질문 생성 중 로딩 표시. `documentAI.generateQuestions()` 호출 |

### 2. 저장 시 기본 문서명이면 이름 변경 유도 (신규)

**이전**: "새 문서"라는 이름 그대로 저장됨
**이후**: 문서명이 "새 문서", "제목 없음", 또는 빈 문자열이면 저장 다이얼로그를 열어 이름을 변경할 기회 제공

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/editor/EditorView.tsx` | `handleQuickSave()`에서 기본 문서명 체크 추가 — 기본값이면 `SaveToProjectDialog` 열림 |

### 3. 저장 흐름 단순화 (v0.1.5에서 이어서)

- 프로젝트만 선택 → 제목+본문 기반 자동 분류 → 카테고리 폴더 자동 생성/선택 → 저장
- 자동 분류 미리보기 표시 ("자동 분류: **보고** 폴더에 저장됩니다")
- 프로젝트 내 폴더 생성 시 디스크에도 반영
