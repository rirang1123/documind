# DocuMind v0.1.9 릴리스 리포트

- **날짜**: 2026-03-10
- **버전**: 0.1.8 → 0.1.9
- **빌드**: DocuMind_0.1.9_x64-setup.exe / DocuMind_0.1.9_x64_en-US.msi

---

## 변경 사항

### 1. 한글 IME 입력 완전 수정

**이전**: `onCompositionEnd`에서 DB 저장 → zustand state 변경 → useEffect가 조합 중 텍스트를 덮어씀 → 스페이스/방향키를 눌러야 완성
**이후**: `isFocusedRef`로 포커스 상태 추적, 포커스 중에는 외부 state 동기화 차단, `onBlur` 시에만 DB 저장

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/settings/SettingsView.tsx` | `useRef`로 포커스 추적, `onCompositionEnd` 제거, `onFocus`/`onBlur` 기반으로 전환 |

### 2. 질문 무한 반복(꼬리물기) 제거

**이전**: 기본 질문 끝나면 AI가 추가 질문 2개 → 또 2개 → 무한 반복
**이후**: 기본 질문 완료 후 수집 정보를 요약 정리하여 보여주고, 추가 정보를 한 번에 입력하거나 바로 생성하는 `confirm` 단계로 전환. 추가 질문 자동 생성 완전 제거.

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/ai/AIAssistant.tsx` | `GenStep`에 `confirm` 추가, `showConfirmStep()` 수집 정보 요약, `handleConfirmSubmit()` 추가 입력/즉시 생성, follow-up 호출 제거, "문서 생성" 버튼 UI 추가 |

### 3. 문서 저장 시 로컬 디스크에 파일 저장 (사용자 선택 포맷)

**이전**: IndexedDB에만 저장, 로컬 디스크에는 파일 없음
**이후**: 프로젝트 폴더 > 분류 폴더 안에 사용자가 선택한 포맷(DOCX/PDF/TXT)으로 실제 파일 저장

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/editor/SaveToProjectDialog.tsx` | 저장 포맷 선택 UI (DOCX/PDF/TXT) 추가, `exportDocument()` 호출로 로컬 디스크 저장, HTML 포맷 제외 |

### 4. HTML 내보내기 포맷 제거 (저장 다이얼로그)

**이전**: 저장 시 HTML 파일로 저장됨
**이후**: DOCX, PDF, TXT 3가지 포맷만 선택 가능, HTML 옵션 제거
