# DocuMind v0.1.12 릴리스 리포트

- **날짜**: 2026-03-10
- **버전**: 0.1.11 → 0.1.12
- **빌드**: DocuMind_0.1.12_x64-setup.exe / DocuMind_0.1.12_x64_en-US.msi

---

## 변경 사항

### 1. 파일 업로드 시 내용 읽기 (빈 파일 버그 수정)

**이전**: 업로드 시 파일 메타데이터만 저장, 내용(`content`/`blobData`)을 읽지 않아 빈 파일로 등록
**이후**: 텍스트 파일(txt, md, html, csv)은 `content`에 텍스트 저장, 바이너리 파일(PDF, PPTX, DOCX 등)은 `blobData`에 ArrayBuffer로 저장

| 파일 | 변경 내용 |
|------|-----------|
| `src/types/index.ts` | `DocumentFile`에 `blobData?: ArrayBuffer` 필드 추가 |
| `src/services/db.ts` | Dexie 스키마 v3 추가 (blobData 지원) |
| `src/components/file-browser/FileBrowser.tsx` | `classifyAndAddFile()`에서 파일 내용 실제 읽기 (텍스트/바이너리 분리) |

### 2. 파일 뷰어 추가 (PDF, 바이너리 파일 지원)

**이전**: 모든 파일 클릭 시 텍스트 에디터로 이동 → 바이너리 파일은 표시 불가
**이후**: 파일 타입별 분기 — 텍스트/HTML 파일은 에디터, PDF는 내장 뷰어(iframe), 기타 바이너리는 다운로드 안내

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/viewer/FileViewer.tsx` | **신규** — PDF 뷰어(iframe), 텍스트 미리보기, 다운로드 기능 |
| `src/stores/useAppStore.ts` | `viewer` 뷰 타입 추가, `viewerFileId`, `openFileViewer()` 추가 |
| `src/components/layout/MainLayout.tsx` | `viewer` 뷰 라우팅 추가 |
| `src/components/file-browser/FileBrowser.tsx` | `handleOpenFile()` — 파일 타입별 에디터/뷰어 분기 처리 |

### 3. 업로드 진행 상태 표시

**이전**: 파일 업로드 시 아무런 피드백 없음
**이후**: 업로드 중 프로그레스 바 + 진행 카운트(n/n) 표시, 업로드 중 "파일 추가" 버튼 비활성화

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/file-browser/FileBrowser.tsx` | `uploading`, `uploadProgress` state 추가, 프로그레스 바 UI |
