# DocuMind v0.4.0 Release Report

**날짜**: 2026-03-11
**버전**: 0.4.0
**주요 변경**: OS Keychain 보안 기반 + 통합 API 관리 + Cloud Provider 추상화

---

## 변경 사항

### Phase 0: OS Keychain 보안 기반

- **Rust**: `keyring@3.6.3` 크레이트 추가 (Windows Credential Store)
- **신규**: `src-tauri/src/keychain.rs`
  - `keychain_set(service, key, value)` — 자격 증명 저장
  - `keychain_get(service, key)` — 자격 증명 조회
  - `keychain_delete(service, key)` — 자격 증명 삭제
- `src-tauri/src/lib.rs` — Tauri invoke handler에 keychain 커맨드 등록
- **신규**: `src/services/credential/keychainService.ts`
  - Tauri invoke 래퍼 + AI/Cloud 전용 헬퍼 함수
- **AIProvider 타입에서 `apiKey` 필드 제거** — Keychain으로 이동
- **DB version 6 마이그레이션**:
  - `cloudProviders`, `activeCloudProviderId` 필드 초기화
  - `migrateApiKeysToKeychain()` — 기존 평문 API 키를 Keychain으로 이동
- **백업 내보내기에서 credential 제외** (apiKey, tokenExpiry)
- **App.tsx**: 초기화 시 `migrateApiKeysToKeychain()` 호출

### Phase 1: 기반 구조

- **신규**: `src/services/credential/tokenManager.ts`
  - `TokenManager` 클래스: access token 유효성 검증 + 자동 갱신
  - mutex 기반 동시 refresh 방지
  - Google OAuth2 token refresh 엔드포인트 연동
- **신규**: `src/services/cloud/cloudProvider.ts`
  - `CloudFile`, `CloudFileList`, `StorageQuota` 인터페이스
  - `CloudStorageProvider` 추상 인터페이스 (authenticate, listFiles, downloadFile, uploadFile, getWebEditUrl 등)
- **타입**: `CloudProvider` 인터페이스 추가 (google-drive | onedrive)
- **AppSettings**: `cloudProviders: CloudProvider[]`, `activeCloudProviderId: string | null` 추가
- **DocumentFile**: `cloudFileId?: string` 필드 추가

### 설정 UI 재구조화

- **SettingsView.tsx**: 탭 "AI 설정" → "API 관리"로 변경
- **API 관리 탭**: AI 서비스 / 클라우드 스토리지 섹션 토글
- **AI 서비스 폼**: Keychain 연동 (저장 시 keychain.setAiApiKey(), 편집 시 keychain.getAiApiKey())
- **클라우드 섹션**: Provider 목록 + 추가/삭제 UI (Phase 3에서 연결 가이드 구현 예정)
- **라디오 버튼**: 활성 AI 제공자 선택 UI 추가

### aiService.ts 변경

- `provider.apiKey` 직접 참조 → `keychain.getAiApiKey(provider.id)` 로 교체
- `getApiKey()` 헬퍼 함수: Keychain에서 API 키 비동기 조회
- 각 provider 메서드에 `apiKey` 파라미터 추가

### CSP 업데이트

- `connect-src`에 Google API 도메인 추가:
  - `https://www.googleapis.com`
  - `https://oauth2.googleapis.com`
  - `https://accounts.google.com`
