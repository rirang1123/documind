# DocuMind v0.4.2 Release Report

**날짜**: 2026-03-11
**버전**: 0.4.2
**주요 변경**: OneDrive 설정 가이드 추가 + 파일 삭제 시 디스크 동기화

---

## 변경 사항

### OneDrive 설정 가이드 추가
- **클라우드 추가 UI 전면 개편**: Google Drive / OneDrive 선택 카드 UI
  - Google Drive: 로고 + "Docs, Sheets, Slides" 서브텍스트
  - OneDrive: 로고 + "Word, Excel, PowerPoint" 서브텍스트
- **신규**: `OneDriveSetupGuide` 컴포넌트 — 4단계 아코디언 가이드
  1. Microsoft Entra ID 앱 등록 (Azure Portal)
  2. API 권한 추가 (Microsoft Graph: Files.ReadWrite, User.Read)
  3. PKCE 설정 확인 (퍼블릭 클라이언트 흐름 허용)
  4. 클라이언트 ID 복사 (애플리케이션 ID)
- 각 단계에 Azure Portal 바로가기 외부 링크
- Client ID 입력 필드 + 저장
- 기존 등록된 클라우드의 "설정" 버튼에서도 provider type에 따라 적절한 가이드 표시

### 파일 삭제 시 디스크 파일 동기화
- **버그 수정**: `useAppStore.deleteFile()` — DB에서만 삭제하고 실제 디스크 파일은 남아있던 문제
- **수정**: `@tauri-apps/plugin-fs`의 `remove()`를 사용하여 `file.path`에 있는 실제 파일도 삭제
- 파일이 없거나 권한이 없는 경우 graceful 처리 (경고 로그만 출력)

### CSP 업데이트
- `connect-src`에 Microsoft 관련 도메인 추가:
  - `https://graph.microsoft.com`
  - `https://login.microsoftonline.com`
