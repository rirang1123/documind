# DocuMind v0.3.4 Release Report

**날짜**: 2026-03-11
**버전**: 0.3.4
**주요 변경**: PPTX 프레젠테이션 미리보기 지원

---

## 변경 사항

### PPTX 미리보기
- `jszip@3.10.1` 설치 — PPTX(ZIP 포맷) 해제용
- **신규**: `src/services/document/pptxService.ts`
  - `parsePptxFromArrayBuffer()` — PPTX 바이너리에서 슬라이드 XML 추출
  - `slidesToHtml()` — 슬라이드별 텍스트를 카드형 HTML로 변환
  - OOXML `<a:p>`, `<a:t>` 태그에서 단락별 텍스트 추출
- **FileViewer.tsx**: pptx blobData → 파싱 → 슬라이드 카드 렌더링, 로딩 스피너

### 뷰어 미리보기 현황
| 확장자 | 미리보기 방식 |
|--------|--------------|
| docx | mammoth → HTML |
| xlsx | SheetJS → HTML 테이블 |
| pptx | JSZip + XML 파싱 → 슬라이드 카드 |
| pdf | iframe |
| txt/md | pre 태그 |
| hwp | 미지원 (의도적 보류) |
