# DocuMind v0.4.1 Release Report

**날짜**: 2026-03-11
**버전**: 0.4.1
**주요 변경**: API 관리 가이드 UI (AI + 클라우드)

---

## 변경 사항

### AI API 키 발급 가이드
- **AI 서비스 편집 폼**: 접이식 API 키 발급 가이드 추가
  - **OpenAI**: platform.openai.com → API keys → Create new secret key
  - **Anthropic**: console.anthropic.com → API Keys → Create Key
  - **Google Gemini**: aistudio.google.com → Get API key → Create API key
- 각 가이드에 **외부 링크 버튼** (해당 사이트 바로 열기)
- `AiApiKeyGuide` 컴포넌트: provider type에 따라 적절한 가이드 표시

### Google Drive 설정 가이드
- **"추가" 버튼** 클릭 시 단계별 설정 가이드 UI 표시 (기존: 빈 provider만 추가됨)
- **4단계 아코디언 가이드**:
  1. Google Cloud 프로젝트 생성
  2. Google Drive API 활성화
  3. OAuth 동의 화면 설정
  4. OAuth 클라이언트 ID 발급
- 각 단계에 **외부 링크 버튼** (Google Cloud Console 해당 페이지 바로 이동)
- **Client ID 입력 필드** + 저장 기능
- 기존 등록된 클라우드에 **"설정" 버튼** 추가 (가이드 재표시)
- `GoogleDriveSetupGuide` 컴포넌트: 신규 등록/기존 편집 모두 지원

### UI 개선
- 새 아이콘 추가: ChevronDown, ChevronRight, ExternalLink, Info
