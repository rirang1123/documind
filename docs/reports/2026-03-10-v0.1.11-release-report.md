# DocuMind v0.1.11 릴리스 리포트

- **날짜**: 2026-03-10
- **버전**: 0.1.10 → 0.1.11
- **빌드**: DocuMind_0.1.11_x64-setup.exe / DocuMind_0.1.11_x64_en-US.msi

---

## 변경 사항

### 1. 업로드 파일 자동 분류 시스템

**이전**: 파일 업로드(드래그&드롭, 파일 추가 버튼) 시 `folderId: null`로 루트에 분류 없이 저장
**이후**: 파일명 + 텍스트 내용(txt, md, html 등 읽기 가능한 파일)을 분석하여 7개 카테고리 중 자동 분류 → 해당 분류 폴더에 자동 배치. 폴더가 없으면 자동 생성.

분류 카테고리: 기획, 평가, 보고, 참고자료, 회의, 계약, 재무, 기타

| 파일 | 변경 내용 |
|------|-----------|
| `src/services/document/classifyService.ts` | **신규** — `detectCategory()`, `ensureCategoryFolder()`, `htmlToPlainText()` 공통 유틸로 분리 |
| `src/components/file-browser/FileBrowser.tsx` | `classifyAndAddFile()` 추가 — 파일 업로드 시 내용 읽기 + 자동 분류 + 폴더 배치 |
| `src/components/editor/SaveToProjectDialog.tsx` | 중복 코드 제거, `classifyService` 공통 모듈 사용으로 전환 |

### 2. 분류 로직 공통화

**이전**: `SaveToProjectDialog` 내부에 `detectCategory`, `ensureCategoryFolder` 함수가 로컬로 정의
**이후**: `classifyService.ts`로 분리하여 파일 브라우저, 저장 다이얼로그 등에서 동일 로직 재사용
