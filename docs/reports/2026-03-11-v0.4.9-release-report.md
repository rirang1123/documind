#### 주요 개선 내용

**1. meeting 키워드 확장**
- 비공식 회의 패턴 추가: `업무 공유`, `팀 공유`, `내용 정리`, `공유 사항`, `금주`, `차주`, `예정 사항` 등
- 구조 패턴에 `논의 후 결정`, `안건 및` 추가
- 제목 접미사에 `회의 안건`, `안건 및 논의` 추가

**2. reference 구조 패턴 강화**
- `작성 방법/원칙/표준`, `양식별 안내/설명`, `활용 팁/노하우` 추가
- 제목 접미사에 `가이드라인`, `절차서`, `guideline` 추가

**3. 충돌 해소 로직 확장 (2개 신규 쌍)**

| 경합 쌍 | 판정 기준 |
|---------|-----------|
| reference ↔ report | 가이드/FAQ/Step 신호 vs 보고서 형식 신호 |
| meeting ↔ evaluation | 회의 형식 신호 + 비공식 회의 패턴 (2개 이상 매칭 시 meeting) |

#### 해결된 오분류 사례
| 문서 제목 | 기존 분류 | 올바른 분류 | 원인 |
|-----------|----------|------------|------|
| "부서 업무 공유 내용 정리" | evaluation | **meeting** | 비공식 회의 패턴 미인식 |
| "팀장 회의 안건 및 논의" | evaluation | **meeting** | "안건" 구조 패턴 부족 |
| "보고서 작성 가이드" | report | **reference** | reference↔report 충돌 해소 없음 |
