# DocuMind v0.3.0 Release Report

**날짜**: 2026-03-11
**버전**: 0.3.0
**주요 변경**: 엑셀(XLSX) 읽기/편집/저장/AI 생성 기능 구현

---

## 변경 사항

### Phase 1: SheetJS 라이브러리 + XLSX 서비스
- `xlsx@0.18.5` (SheetJS) 라이브러리 설치
- **신규**: `src/services/document/excelService.ts`
  - `parseExcelFromArrayBuffer()` - XLSX 바이너리 파싱
  - `parseExcelFromContent()` - CSV/텍스트 파싱
  - `generateExcelBuffer()` - SheetData → XLSX 바이너리 생성
  - `sheetToHtmlTable()` - HTML 테이블 변환 (뷰어용)

### Phase 2: SpreadsheetEditor 고도화
- **파일**: `src/components/editor/spreadsheet/SpreadsheetEditor.tsx`
  - `initialSheets` prop으로 파싱된 멀티시트 데이터 수신
  - 시트 탭 UI (하단 탭으로 시트 전환)
  - 새 시트 추가 (+) 버튼
  - 숫자 셀 우측 정렬
  - `onChange` 콜백으로 SheetData[] 전달

### Phase 3: EditorView + FileBrowser + FileViewer 연동
- **EditorView**: xlsx 파일 열기 시 `parseExcelFromArrayBuffer()` 자동 호출, SpreadsheetEditor에 데이터 전달, 저장 시 `generateExcelBuffer()` → DB blobData 저장
- **FileBrowser**: xlsx 파일 클릭 시 뷰어 대신 에디터(SpreadsheetEditor)로 라우팅
- **FileViewer**: xlsx 파일 미리보기 시 HTML 테이블로 렌더링 (DOMPurify 적용)

### Phase 4: AI 스프레드시트 생성 + Export
- **documentAI.ts**: `generateSpreadsheet()` 함수 추가 (AI가 JSON으로 데이터 생성 → SheetData 변환)
- **AIAssistant.tsx**: "스프레드시트" 모드 추가 (주제 입력 → AI 생성 → xlsx 파일로 프로젝트에 저장 → 에디터로 열기)
- **exportService.ts**: `ExportFormat`에 'xlsx' 추가, `exportXlsx()` 함수
- **ExportDialog.tsx**: Excel(.xlsx) 내보내기 옵션 추가, 스프레드시트 에디터에서 xlsx 기본 선택

### 기타
- **SaveToProjectDialog.tsx**: ExportFormat Record에 xlsx 매핑 추가
