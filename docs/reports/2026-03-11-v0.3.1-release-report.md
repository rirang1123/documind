# DocuMind v0.3.1 Release Report

**날짜**: 2026-03-11
**버전**: 0.3.1
**주요 변경**: 파일 열기 아키텍처 전환 — 뷰어 중심 + 외부 앱 편집

---

## 아키텍처 변경 요약

### 이전 (v0.3.0)
- 파일 클릭 → 확장자별 내장 에디터(DocumentEditor, SpreadsheetEditor, PresentationEditor)로 열기
- DocuMind이 모든 파일 형식의 편집기 역할

### 이후 (v0.3.1)
- **파일 클릭 → 뷰어(FileViewer)로 열기** (읽기 전용 미리보기)
- **"편집하기" 버튼 → OS 기본 앱으로 열기** (Excel, Word, PowerPoint 등)
- **PDF는 편집 버튼 없음** (뷰어에서 열람만)
- **AI 생성 문서 → 내장 TipTap 에디터** (1차 편집 후 저장 → 이후 뷰어로 취급)
- **AI 생성 스프레드시트 → xlsx로 저장 후 뷰어로 열기**

### 핵심 원칙
1. AI가 생성한 문서는 내장 에디터에서 1차 편집 가능
2. 저장되어 확장자가 확정된 파일은 뷰어로 열람
3. PDF 제외 모든 파일에 "편집하기" 버튼 → 외부 앱 실행

## 기술 세부사항

### 외부 앱 열기 흐름
1. 파일에 원본 경로가 있으면 → `openPath(filePath)` 직접 실행
2. DB에 blobData만 있으면 → `$APPDATA/temp/파일명`에 임시 저장 → `openPath(tempPath)`
3. text content만 있으면 → 동일하게 임시 파일 저장 후 열기

### 뷰어 미리보기 지원
- **xlsx**: SheetJS 파싱 → HTML 테이블 (DOMPurify)
- **PDF**: iframe
- **txt/md**: pre 태그
- **HTML content (docx 등)**: DOMPurify sanitize → prose 렌더링
- **기타 바이너리**: 파일명 + 크기 표시 + 편집/다운로드 버튼
