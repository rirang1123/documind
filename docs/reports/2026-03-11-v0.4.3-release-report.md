# DocuMind v0.4.3 Release Report

**날짜**: 2026-03-11
**버전**: 0.4.3
**주요 변경**: 코드 리뷰 기반 보안/안정성 수정 (33개 이슈 중 주요 항목 수정)

---

## 변경 사항

### CRITICAL 수정

#### 1. FileViewer Blob URL 메모리 누수 수정
- **파일**: `src/components/viewer/FileViewer.tsx`
- **문제**: `useMemo`로 생성된 Blob URL이 해제되지 않아 메모리 누수 발생
- **수정**: `useEffect` + `URL.revokeObjectURL()` cleanup으로 변경

#### 2. AIAssistant 아웃라인 XSS 취약점 수정
- **파일**: `src/components/ai/AIAssistant.tsx`
- **문제**: `dangerouslySetInnerHTML={{ __html: outline }}` — DOMPurify 미적용
- **수정**: `DOMPurify.sanitize(outline)` 적용

#### 3. TokenManager OneDrive 지원 + per-provider mutex
- **파일**: `src/services/credential/tokenManager.ts`
- **문제**: 토큰 갱신 엔드포인트가 Google OAuth에 하드코딩, 단일 mutex로 다른 provider 갱신 블로킹
- **수정**: `provider.type`에 따라 Google/Microsoft 엔드포인트 분기, `Map<string, Promise>` per-provider mutex

#### 4. deleteFolder 고아 파일 방지
- **파일**: `src/stores/useAppStore.ts`
- **문제**: 폴더 삭제 시 하위 파일들이 참조 없는 고아 상태로 남음
- **수정**: 폴더 삭제 전 하위 파일들을 루트(`folderId: null`)로 이동

#### 5. UpdateChecker 데드코드 제거
- **파일**: `src/components/layout/UpdateChecker.tsx`, `MainLayout.tsx`
- **문제**: updater 플러그인이 Rust 백엔드에 등록되지 않아 항상 실패하는 데드코드
- **수정**: `UpdateChecker` 컴포넌트 참조 제거, `@tauri-apps/plugin-updater` 의존성 제거

### MAJOR 수정

#### 6. Header 검색 결과에서 바이너리 파일 처리
- **파일**: `src/components/layout/Header.tsx`
- **문제**: PDF/DOCX/XLSX/PPTX 파일 클릭 시 에디터로 열어 빈 화면 표시
- **수정**: 바이너리 파일 타입은 `openFileViewer()`로 뷰어에서 열기

#### 7. Header 검색 드롭다운 다크모드 호환
- **파일**: `src/components/layout/Header.tsx`
- **문제**: `bg-white` 하드코딩으로 다크모드에서 배경색 깨짐
- **수정**: `bg-background`로 변경 (테마 토큰 사용)

#### 8. DB 백업/복원에 aiUsage, templates 테이블 포함
- **파일**: `src/services/db.ts`
- **문제**: `exportDatabaseToJSON()`에서 aiUsage, templates 테이블 누락
- **수정**: export/import 모두 aiUsage, templates 포함

#### 9. 백업 버전 번호 수정
- **파일**: `src/services/db.ts`
- **문제**: `backup.version`이 4로 하드코딩 (실제 DB 스키마 버전은 6)
- **수정**: `version: 6`으로 수정

#### 10. ExportService innerHTML XSS 방지
- **파일**: `src/services/document/exportService.ts`
- **문제**: `htmlToText()`, `parseHtmlBlocks()`, `exportPdf()`, `exportHtml()`에서 DOMPurify 미적용
- **수정**: 모든 innerHTML 사용 시 `DOMPurify.sanitize()` 적용, HTML export 제목 이스케이프

#### 11. package.json 의존성 정리
- `@tauri-apps/cli`를 dependencies → devDependencies로 이동
- `@tauri-apps/plugin-updater` 제거

### MINOR 수정

#### 12. EditorView shortcut:save useEffect 의존성 배열 누락
- **파일**: `src/components/editor/EditorView.tsx`
- **문제**: useEffect에 의존성 배열 없어 매 렌더마다 listener 재등록
- **수정**: `useRef` 패턴으로 안정적인 handler + `[]` 의존성 배열
