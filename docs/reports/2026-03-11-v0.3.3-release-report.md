# DocuMind v0.3.3 Release Report

**날짜**: 2026-03-11
**버전**: 0.3.3
**주요 변경**: DOCX 파일 미리보기 지원 (mammoth 라이브러리)

---

## 변경 사항

### DOCX 미리보기
- `mammoth@1.11.0` 라이브러리 설치 — docx 바이너리를 HTML로 변환
- **FileViewer.tsx**: docx blobData → `mammoth.convertToHtml()` → DOMPurify sanitize → 렌더링
- 로딩 중 스피너 표시
- AI 에디터에서 저장한 docx(content 필드)도 HTML 미리보기 지원

### 뷰어 미리보기 현황
| 확장자 | 미리보기 방식 |
|--------|--------------|
| docx | mammoth → HTML 렌더링 |
| xlsx | SheetJS → HTML 테이블 |
| pdf | iframe |
| txt/md | pre 태그 |
| pptx | 미지원 (편집하기/다운로드) |
