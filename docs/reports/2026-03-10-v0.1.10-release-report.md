# DocuMind v0.1.10 릴리스 리포트

- **날짜**: 2026-03-10
- **버전**: 0.1.9 → 0.1.10
- **빌드**: DocuMind_0.1.10_x64-setup.exe / DocuMind_0.1.10_x64_en-US.msi

---

## 변경 사항

### 1. 파일시스템 권한 전면 확장

**이전**: FS 권한이 `$DOCUMENT` 폴더에만 제한 → 사용자가 바탕화면, 홈 폴더 등에 프로젝트를 만들면 파일 저장 실패
**이후**: `$HOME`, `$DESKTOP`, `$DOWNLOAD`, `$APPDATA`, `$LOCALAPPDATA` 경로 전체에 읽기/쓰기 권한 부여. `fs:scope`로 통합 관리.

| 파일 | 변경 내용 |
|------|-----------|
| `src-tauri/capabilities/default.json` | 개별 권한의 `allow` 경로 제한 제거, `fs:scope`로 통합하여 다양한 경로 허용 |

### 2. 로컬 디스크 저장 경로 수정

**이전**: 문자열 조합(`${path}/${folder}`)으로 경로 생성 → Windows 경로 구분자 불일치 가능
**이후**: Tauri `join` API 사용으로 OS별 올바른 경로 생성

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/editor/SaveToProjectDialog.tsx` | `@tauri-apps/api/path`의 `join` 사용으로 경로 조합 수정 |
