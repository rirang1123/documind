# DocuMind v0.1.13 릴리스 리포트

- **날짜**: 2026-03-10
- **버전**: 0.1.12 → 0.1.13
- **빌드**: DocuMind_0.1.13_x64-setup.exe / DocuMind_0.1.13_x64_en-US.msi

---

## 변경 사항

### 1. AI 문서 생성 — 확인 단계 버그 수정

**이전**: 확인 단계에서 Enter 키로 빈 입력 전송이 차단되어 "생성" 버튼만 사용 가능
**이후**: Enter 키로도 바로 문서 생성 가능 (빈 입력 = 바로 생성)

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/ai/AIAssistant.tsx` | `handleSend` 가드에 `genStep === 'confirm'` 조건 추가 |

### 2. AI 문서 생성 — 오류 복구 단계 수정

**이전**: 문서 생성 중 오류 발생 시 `'asking'`(질문) 단계로 돌아감 — 이미 완료된 질문을 다시 진행하게 됨
**이후**: `'confirm'`(확인) 단계로 돌아가서 바로 재생성 가능

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/ai/AIAssistant.tsx` | `startGenerate` 에러 핸들러에서 `setGenStep('confirm')` 으로 변경 |
