# DocuMind v0.3.2 Release Report

**날짜**: 2026-03-11
**버전**: 0.3.2
**주요 변경**: 엑셀 미리보기 — 데이터 있는 셀까지만 표시

---

## 변경 사항

### excelService.ts
- `trimData()` 함수 추가: 후행 빈 행/열을 제거하여 데이터가 있는 범위만 반환
- `sheetToHtmlTable()`에서 `trimData()` 적용: 빈 셀이 끝없이 나오지 않고 실제 데이터 범위만 렌더링
