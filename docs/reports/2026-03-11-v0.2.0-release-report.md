# DocuMind v0.2.0 Release Report

**Release Date**: 2026-03-11
**Previous Version**: 0.1.13
**Version Jump**: 0.1.13 → 0.2.0 (Major Feature Release)

---

## Summary

CTO 고도화 계획 Phase A~E를 한 번에 구현한 대규모 릴리스입니다.
보안 취약점 5건 수정, UX 개선 4건, 안전망 3건, 파일 관리 6건, AI 차별화 기능 6건을 포함합니다.

---

## Phase A: 보안 + 안정성 (5건)

### A1. XSS 방어 — DOMPurify 적용
- **파일**: `src/components/viewer/FileViewer.tsx`
- `dangerouslySetInnerHTML`에 `DOMPurify.sanitize()` 적용
- dompurify + @types/dompurify 설치

### A2. CSP 활성화
- **파일**: `src-tauri/tauri.conf.json`
- `"csp": null` → 엄격한 CSP 정책 설정
- AI API 도메인(OpenAI, Anthropic, Google)만 connect-src 허용

### A3. Error Boundary 추가
- **파일**: 신규 `src/components/ErrorBoundary.tsx`, `src/App.tsx`
- 컴포넌트 크래시 시 "오류 발생" 폴백 UI + 다시 시도/새로고침 버튼
- MainLayout을 ErrorBoundary로 래핑

### A4. App 초기화 에러 처리
- **파일**: `src/App.tsx`
- loadSettings/loadProjects/loadDrafts에 try-catch 적용
- 초기화 실패 시 에러 메시지 + 재시도 UI 표시

### A5. 삭제 확인 다이얼로그
- **파일**: `src/components/file-browser/FileBrowser.tsx`
- 파일/폴더 삭제 시 2단계 확인 (deleteTarget 패턴)
- "삭제하시겠습니까?" 다이얼로그 + 취소/삭제 버튼

## Phase B: UX 개선 (4건)

### B1. 다크모드 구현
- **파일**: `src/index.css`, `src/App.tsx`
- `.dark` 클래스 CSS 변수 오버라이드 추가
- settings.theme 연동 (light/dark/system 지원)

### B2. 헤더 검색 구현
- **파일**: `src/components/layout/Header.tsx`, `src/stores/useAppStore.ts`
- 파일명 + 내용 검색, 결과 드롭다운 표시
- 클릭 시 파일 열기, Escape/외부 클릭으로 닫기

### B3. 키보드 단축키
- **파일**: 신규 `src/hooks/useKeyboardShortcuts.ts`, `src/App.tsx`
- Ctrl+S (저장), Ctrl+N (새 문서), Escape (검색 닫기)

### B4. 에디터 디바운스
- **파일**: `src/components/editor/EditorView.tsx`
- setEditorContent에 300ms 디바운스 적용
- 타이핑 중 불필요한 전체 리렌더 방지

## Phase C: 안전망 (3건)

### C1. 데이터 백업/복원
- **파일**: `src/services/db.ts`, `src/components/settings/SettingsView.tsx`
- 설정에 "백업"/"복원" 버튼 추가

- 복원 시 확인 다이얼로그 포함

### C2. 업데이터 비활성화
- **파일**: `src-tauri/tauri.conf.json`, `src-tauri/src/lib.rs`, `src-tauri/Cargo.toml`
- YOUR_USERNAME 플레이스홀더 + 빈 pubkey 제거
- tauri-plugin-updater 의존성 완전 제거

- `$HOME/**` 제거 → `$DOCUMENT`, `$DESKTOP`, `$DOWNLOAD`, `$APPDATA`, `$LOCALAPPDATA`만 유지

### E2. AI 문서 아웃라인 생성
- **파일**: `src/components/ai/AIAssistant.tsx`, `src/services/ai-provider/documentAI.ts`

- generateOutline() 함수로 목차 구조 먼저 제안

### E3. 문서 템플릿 시스템
- **파일**: `src/services/db.ts` (templates 테이블), `src/services/document/builtinTemplates.ts` (신규)
- **파일**: 신규 `src/components/editor/TemplateManager.tsx`, `src/components/editor/NewDocumentDialog.tsx`
- 내장 템플릿 6개 (업무 보고서, 회의록, 기획서, 공식 이메일, 주간 보고서, 이력서)
- 현재 문서를 템플릿으로 저장, 새 문서에서 템플릿 선택

### E4. 문서 비교 (Diff) 뷰
- **파일**: 신규 `src/components/viewer/DiffViewer.tsx`
- 두 문서 나란히 비교, 추가/삭제 하이라이트

### E5. AI 비용 추적 대시보드
- **파일**: `src/services/db.ts` (aiUsage 테이블), `src/services/ai-provider/aiService.ts`
- **파일**: `src/components/settings/SettingsView.tsx`
- AI 요청마다 토큰 수 기록, 월별 사용량 통계

### E6. 프로젝트 통계 대시보드
- **파일**: 신규 `src/components/project/ProjectDashboard.tsx`
- **파일**: `src/stores/useAppStore.ts`, `src/components/layout/MainLayout.tsx`, `src/components/layout/Sidebar.tsx`
- 파일 수, 용량, 타입 분포, 최근 활동 차트
