DocuMind 개인정보처리방침

시행일: 2026년 3월 13일

1. 개요

DocuMind는 AI 기반 문서 관리 데스크톱 애플리케이션입니다. 본 방침은 DocuMind가 사용자 정보를 어떻게 처리하는지 설명합니다.

2. 수집하는 정보

DocuMind는 서버를 운영하지 않으며, 사용자 데이터를 외부로 전송하거나 수집하지 않습니다.

  • 문서 데이터: 모든 문서는 사용자의 로컬 디스크에만 저장됩니다.
  • API 키: 사용자가 직접 입력한 AI 서비스 API 키는 OS Keychain(Windows Credential Store)에 암호화되어 저장되며, 외부로 전송되지 않습니다.
  • Google Drive 연동: 사용자가 직접 생성한 OAuth 자격증명을 사용하며, DocuMind 개발자는 사용자의 Google 계정에 접근할 수 없습니다.

3. 제3자 서비스

DocuMind는 사용자가 직접 설정한 경우에만 다음 서비스와 통신합니다:

  • AI API (OpenAI, Anthropic, Google AI): 문서 생성 및 분류를 위해 사용자가 입력한 API 키로 요청을 전송합니다.
  • Google Drive API: 사용자가 직접 OAuth 인증한 경우에만 파일 가져오기/내보내기에 사용됩니다.

각 서비스의 데이터 처리는 해당 서비스 제공자의 개인정보처리방침을 따릅니다.

4. 데이터 저장

  • 모든 데이터는 사용자의 로컬 컴퓨터에만 저장됩니다.
  • 앱 삭제 시 모든 데이터를 함께 삭제할 수 있습니다.
  • 백업/복원 기능은 로컬 파일로만 동작합니다.

5. 분석 및 추적

DocuMind는 사용자 행동 분석, 추적, 원격 측정(telemetry)을 일체 수행하지 않습니다.
